Context and Project Definition

In a context where awareness of the importance of health is constantly growing, the need for consumers to easily access reliable and understandable information about the products they consume has become paramount. This project aims to meet this requirement by developing a digital solution that allows scanning of food product labels, thus offering an instant analysis of their composition. This project aims to make product information transparent and accessible to all, contributing to improving quality of life.

Project Objective

The main objective of this project is to develop a mobile application that facilitates access to detailed and reliable information on the composition of food products, in order to improve consumers' decision-making regarding their health. To achieve this overall objective, the project focuses on three specific objectives:

  • Designing an accessible and intuitive user interface: The objective is to enable a smooth and pleasant user experience, thereby reducing the learning curve and increasing user engagement.
  • Integrating a product search function: Developing a feature that allows users to instantly access detailed information about a given product.
  • Providing ingredient analysis: Offering users clear and accurate information about ingredients. The goal is to help consumers understand the impact of their dietary choices on their health and make informed decisions about their diet.

Project Scope

The scope of this project includes the development of a mobile application dedicated to promoting healthy and informed eating for consumers.

Functional Description of Requirements

Main Features

  • Accessible and Intuitive User Interface: Creation of a user interface (UI) that is easy to navigate. The UI should be designed to be aesthetically pleasing and functional on various mobile devices.
  • Product Search: Offering a search function that allows users to find products by name, category, or brand. The search can be performed through barcode scanning for offline usage.
  • Ingredient Analysis: Providing detailed and reliable information on the composition of food products, including nutritional values, ingredients, and allergens.
  • Share Results: Allowing users to share the analysis results with their contacts and social networks.
  • Offline Product History: Storing a history of previously viewed products for offline access, enabling users to access product information even without an internet connection.

Secondary Features

  • User Profile Creation: Allowing users to create profiles, including registration, login, and account management.
